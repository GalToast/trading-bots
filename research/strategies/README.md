# Strategy Prototypes

Strategy prototypes and Spark/OANDA/Alpaca experiment families live here.

Several byte-for-byte duplicate files were removed from the repo root during the 2026-04-30 GitHub presentation cleanup. This directory is the intended home for those prototype families.

Broker credentials are intentionally not included. Strategies that reference Alpaca or OANDA expect local environment-backed config modules or environment variables; they are included as reviewable prototypes, not as one-command public demos.
